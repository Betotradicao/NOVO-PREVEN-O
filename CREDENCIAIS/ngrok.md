# Credenciais do Ngrok

## Token de Autenticação

**Arquivo de configuração**: `InstaladorINTERNO/ngrok.yml`

Substitua `YOUR_NGROK_TOKEN_HERE` pelo seu token real do Ngrok.

## Como obter o token:

1. Acesse: https://dashboard.ngrok.com/
2. Faça login ou crie uma conta
3. Vá em "Your Authtoken"
4. Copie o token
5. Cole no arquivo `InstaladorINTERNO/ngrok.yml`

## Túneis Configurados:

- **Frontend**: http → porta 3004
- **Backend**: http → porta 3001
